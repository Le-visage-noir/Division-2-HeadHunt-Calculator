A modular "Head Hunt" build optimization tool for Tom Clancy's The Division 2, supporting combinatorial gear generation, damage modeling, constraint-based filtering, and ranking.

Includes a lightweight Streamlit-based web interface for interactive analysis across devices.


一個用於《湯姆克蘭西：全境封鎖2》「堅定獵頭」配裝優化的計算工具，提供裝備組合生成、傷害估算、條件篩選與排序功能。使用者可自訂武器、屬性與加成，快速找出最優配置。

本專案採用模組化設計，並提供簡易網頁介面，方便在不同裝置上進行分析與比較。
